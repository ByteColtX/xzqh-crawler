# 行政区划代码爬虫项目需求文档

## 项目概述
开发一个爬虫系统，用于从民政部网站（dmfw.mca.gov.cn）获取行政区划代码数据。

## 数据源分析
- **目标网站**: https://dmfw.mca.gov.cn/XzqhVersionPublish.html
- **API端点**: https://dmfw.mca.gov.cn/xzqh/getList（获取level 1~3数据）
- **数据格式**: JSON格式的树形结构（包含code、name、level、type、children字段）
- **数据层级**: 省、地、县三级行政区划(level: 1~3)，乡级数据需要其他方式获取
- **重要发现**: API返回树形结构，不包含year、parentCode、parentName、namePath字段，需要从树形结构推导

## 核心需求

### 1. 数据获取需求
- [ ] 能够获取完整的行政区划代码树
- [ ] 支持按层级获取数据（省、地、县、乡）
- [ ] 支持按行政区划代码获取下级数据
- [ ] 支持批量获取数据

### 2. 数据存储需求
- [ ] 存储在SQLite


### 3. 性能需求
- [ ] 支持多线程或异步请求
- [ ] 支持并发请求
- [ ] 支持断点续传
- [ ] 支持缓存机制
- [ ] 支持错误重试

### 4. 用户体验需求
- [ ] 配置化管理
- [ ] 进度显示

## 不做
- 不做API接口的开发
- 不做命令行接口
- 不做数据导出和统计功能
